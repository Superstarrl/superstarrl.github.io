Project Summary

Mechanisoul is an action-combat prototype centered around a mechanical being discovering its own consciousness through movement, combat, and systems interaction. The project is designed as a long-term sandbox where core systems—movement, combat feel, AI behavior, and progression—can be iterated on and reused across future titles.

Intent & Current State

The goal is to create a fast yet readable combat system with strong feedback, precise movement, and expressive decision-making. Current progress includes:

  • Prototype character movement controller
  • Early combat animations and hit detection
  • Basic enemy behavior for spacing and timing tests
  • Camera behavior tuned for readability

What I Have Learned So Far

  • Prototyping animation rigs for responsive control
  • Improving iteration workflows for tuning movement/attack timing
  • Separating input, movement, and combat logic into reusable modules
  • Scoping vertical-slice goals for an evolving long-term project

Systems Breakdown

  • Character Controller: Early controller focused on grounded feel and camera integration
  • Combat Prototyping: Basic attack logic, hit detection, spacing tools
  • AI Foundations: Dummy targets and placeholder AI for timing tests
  • Progression Concepts: Future upgrade & modular systems planned
  • Debug Tools: Custom testing environments for movement/spacing/value tuning
