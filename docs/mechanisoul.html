Mechanisoul (In Development)

Project Summary

Mechanisoul is an in-progress action game about a mechanical being discovering its own consciousness through combat, exploration, and interaction with a world built from modular machines. The focus of the project is on responsive character control, readable combat behavior, and systems that support expressive player choices.

Mechanisoul is being built as a long-term sandbox for testing character controllers, AI behavior, and progression systems that I can reuse and expand across multiple projects. The current phase is focused on solidifying core movement, combat feel, and iteration tools before building full levels or narrative content.

Intent & Current State

The intention is to create a tightly scoped combat experience with a strong sense of weight, impact, and clarity. Right now, the game includes an early character model, baseline movement, and prototype combat interactions. Some systems are still placeholder or blocked out, and the visuals are expected to change as the mechanics stabilize. The page will continue to evolve as new milestones are reached.

What I Have Learned So Far

  • How to prototype character rigs and animations to support responsive control.
  • Better workflows for iterating on movement speeds, turn rates, and timing windows.
  • Improved structure for separating input, movement, and combat logic for reuse.
  • Early lessons in scoping vertical slice goals for an ongoing project.
