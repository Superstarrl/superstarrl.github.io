Project Summary

Mechanisoul is an action-combat prototype centered around a mechanical being discovering its own consciousness through movement, combat, and systems interaction. The project is designed as a long-term sandbox where core systems—movement, combat feel, AI behavior, and progression—can be iterated on and reused across future titles.

Intent

The goal is to create a difficult, yet rewarding, combat system with a variety of upgrades and combos that create a very replayable experience

What I Have Learned So Far

  • Improvements in designing and building Modular System Frameworks that Allow for flexible implementation
  • Improved Communication when working with a larger coding team

My Contributions

  • Upgrade Framework: Framework for creating and implementing Player Upgrade Systems
  • Melee Upgrades: Implementation of Melee Upgrade System
  • UI Integration: Link all Upgrade Components to their UI counterparts
